# Web app

Next.js application for IncidentMind.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Prisma ORM (PostgreSQL)

## Status

- App Router, layout, and placeholder pages for Dashboard, Incident
  Details, Incident History, Analytics, and Settings are in place.
- No business logic, API routes, or database schema have been added
  yet — those will be added in subsequent iterations.

## Run

```bash
pnpm dev:web
```

The app listens on `http://localhost:3000` by default.
